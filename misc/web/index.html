Structure Synth

Structure Synth is an application for creating 3D structures from a set of user specified rules. It is an attempt to make a 3D version of Context Free.

It is built in C++, OpenGL, and Qt 4.3. Builds will (at some point) be provided for Windows, Linux, and Mac.

As of now this project is in prototyping phase, and this web page is a temporary placeholder. My progress and design considerations can be followed at my Syntopia blog.

 

The source code is hosted on SourceForge, and can be accessed using SVN:

svn co https://structuresynth.svn.sourceforge.net/svnroot/structuresynth/trunk structuresynth
